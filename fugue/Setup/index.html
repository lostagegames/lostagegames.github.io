# Set Up

For 2 player games

< Figure 1 image >

  1. Each player places one Player Board (A) in front of them. The Deck Board (B) is placed in between the Player Boards.

  2. Shuffle the cards into a face-down deck on the Deck Board (C).

  3. Place the fugue meeples near the Deck Board (D).

  4. Each player places a Score Cube on the “0” of the Score Track on their own Player Board (E).

For a 1 player game

Fugue has an exciting solitaire mode! See the section on solo play on page 12 for more details.

*For 3+ player games

For a 3 or 4 player game, add one additional game-set. For a 5 or 6 player game, add two additional game-sets, etc.

It’s possible to have any number of players as long as new game-sets are added, but expect games to last an extra 20 minutes for each player over two.
