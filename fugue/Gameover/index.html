# Game Over

After the 4th Spirit card has been removed from the game, the game is over the next time the deck becomes empty.

# Victory

When the game is over, add or subtract the value of Fugues the player’s possess from their Score Track.

The Player with the most points wins!

# Ties

The player who started last wins ties.
