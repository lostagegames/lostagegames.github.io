# Game Modes

Fugue can be expanded to include more than two players by simply adding additional game sets.

Fugue also has two Solo Modes, Left Brain vs Right Brain, and Solo Challenge.

Standard Fugue is the game mode described throughout the rulebook. Different game modes have slight rules differences.

# 3-4 player game

No Players: 3-4 Play time: 40 - 60 min Requires: 2 game sets

Plays the same as Standard Fugue, but uses an extra game set. Each player uses their own Player Board.

Rules Change: The game is over after 2 Spirit cards have been removed from the game and the deck becomes empty again (instead of the standard 4 Spirit cards).

# 5-6 player game

No Players: 5-6 Play time: 80 - 100 min Requires: 3 game sets

Plays the same as Standard Fugue, but with two extra game sets. Each player uses their own Player Board.

Rules Change: The game is over after 1 Spirit card has been removed from the game and the deck becomes empty again (instead of the standard 4 Spirit cards).

# Chaos Fugue (play at your own risk!)

No Players: 7+ Play time: 100+ min Requires: 4+ game sets

This plays the same as Standard Fugue, but follows the pattern of adding an extra game set for every 2 additional players. Each player uses their own Player Board.

Expect a long game!

Rules Change: The game is over after the deck becomes empty once (no removing Spirit cards or reshuffling the Depth Area back into the deck!).
